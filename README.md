# Ford Bike Sharing Analytics Dashboard

## Project Overview
This project analyzes the Ford Bike Sharing Dataset using Exploratory Data Analysis (EDA) and Machine Learning.

## Features
- Home Dashboard
- EDA Dashboard
- User Type Prediction
- Business Insights
- Business Recommendations
- Conclusion

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn
- Joblib

## Machine Learning Model
Random Forest Classifier

## Run the Project
```bash
cd streamlit_app
streamlit run app.py
```
